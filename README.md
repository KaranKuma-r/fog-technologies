<h2> Features</h2>

<h3> Game Selection Interface</h3>
<ul>
  <li>🎮 Interactive game cards</li>
  <li>🖱 Mouse drag + 📱 touch swipe support</li>
  <li>🎥 Dynamic video preview (different video per game)</li>
  <li>🌫 Blur background with smooth UI</li>
  <li>⬅️➡️ Navigation buttons</li>
  <li>⚡ Smooth transitions & animations</li>
</ul>

<h3>🎯 Grid Game</h3>
<ul>
  <li>🔢 Dynamic grid (rows & columns)</li>
  <li>🎯 Pattern 1 & Pattern 2</li>
  <li>🔄 Auto + Manual pattern switching</li>
  <li>🔵 Blue tiles → Collect points</li>
  <li>🔴 Red tiles → Blink + lose life</li>
  <li>🟢 Green tiles → Safe zone</li>
  <li>❤️ Lives system (5 lives)</li>
  <li>⏱ Timer (30 seconds)</li>
  <li>🏆 Win / Lose conditions</li>
  <li>🎨 Glass UI + Neon effects</li>
</ul>

<h2>🎮 How to Use</h2>

<ul>
  <li>Swipe or drag to navigate games</li>
  <li>Click on a game card → watch preview video</li>
  <li>Click "Start Game"</li>
  <li>Play grid challenge</li>
  <li>Collect all 🔵 tiles before time ends</li>
  <li>Avoid 🔴 tiles (lose life)</li>
</ul>
